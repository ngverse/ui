Pipes module includes collection of pipes that are commonly used in the Angular application.

Schematic Usage

Options

  • path: Specifies where to place the component folder.
  • project: The name of the Angular project where the component should be added. By default it will use defaultProject
  • replace: Determines whether to replace existing files with the same name. False by Default
